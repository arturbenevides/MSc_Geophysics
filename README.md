# Magnetotelúrico

Repository, DSC ON (1)

# Imageamento Magnetotelúrico 3D da bacia do Iguatu: implicações hidrogeológicas. 


*O diretório Iguatu contém informações sobre a área de estudo onde é empregado o método MT.
